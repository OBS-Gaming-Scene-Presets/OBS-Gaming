# OBS Gaming Scene Presets
Community-contributed scene collections, overlay templates and profiles optimized for gaming capture and streaming in **OBS Studio**.

> These are user-made presets only.  
> Always use the **official** OBS Studio from https://obsproject.com

![OBS Studio Interface](https://static0.howtogeekimages.com/wordpress/wp-content/uploads/2025/07/a-streamer-wearing-headphones-sits-in-front-of-a-microphone-with-obs-studio-interface-elements-overlaid-on-the-image-and-the-obs-logo-surrounding-them.png?w=1600&h=900&fit=crop)

## What's inside this repository
- 🎮 Gaming-focused scene collections (.json / .scn files)  
- 🖼️ Simple overlay templates (Browser/HTML/CSS sources)  
- 🎧 Basic audio filter & transition presets  
- ⚡ Performance-oriented profiles (for mid/low-end PCs)  
- 📸 Example screenshots and usage notes  

All files are plain text / images — **no executables, no installers, no modifications** to OBS itself.

## How to use
1. Download the latest release from the **Releases** tab on the right → choose the .zip you need  
2. Extract the archive to any convenient folder  
3. Open official **OBS Studio** (download from https://obsproject.com if needed)  
4. Go to **Scene Collection** → **Import** → select the .json / .scn file from the extracted folder  
5. (Optional) For profiles: **Profile** → **Import** → select .json profile file  
6. Adjust sources (Game Capture, Webcam, Audio) to match your hardware  

**Tip**: Create a backup of your current scenes before importing new ones (Scene Collection → Duplicate).
